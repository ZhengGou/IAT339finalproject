Interaction web Design: The maid

Introduction

Spring 2019 - As a team of four in this Interaction web Design course, we are presenting our final term project named "The maid". The maid is a interaction web base text explore game which the targeted audiences are both males and females aged between 15 to 45, who are interesting with spend some time on text-image game. The main idea of the website is to allow user to have immersion experience by interaction with the image.

My role: visual designer, interaction designer. I create the flow chart of interaction.

Process

Create setting

In the beginning, the professor asked us to think about the character set. We found that we were very interested in this role-playing case. Through discussion, we decided to make this text-based web page. First of all, we have drawn a room diagram to make it easier for us to combine the script with this idea.

We also created a moodboard on the printerest to let the professor understand the feeling of making our web page.

Process

I made a flow chart based on the script, and I used the way of illustration to illustrate the logic of our web page exploring interaction. Because we decided to make this web game on wix, I found the corresponding icon to display the items that can be clicked. I think the visual expression helps my team members to make and improve the story line more stereoscopically.

We also created a flowchart of the storyboard version to get a more intuitive understanding of how the story interacts.
