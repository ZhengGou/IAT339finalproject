Interaction web Design: The maid

Introduction

Spring 2019 - As a team of four in this Interaction web Design course, we are presenting our final term project named "The maid". The maid is a interaction web base text explore game which the targeted audiences are both males and females aged between 15 to 45, who are interesting with spend some time on text-image game. The main idea of the website is to allow user to have immersion experience by interaction with the image.

My role: visual designer, interaction designer. I create the flow chart of interaction.
