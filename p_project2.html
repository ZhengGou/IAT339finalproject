01

The Clock

This is a project in the Spatial Design course. This project requires us to create a "Cube-in-a-box", a personal space for reading, reflection, relaxation and rest. We need to create a 50x50x50 cm object cube where people can sit and put things on it. However, the box does not have to be a box, and the cube does not have to be a cube. The project aims to deepen the understanding of how to combine human size, function and space formation technology in space design.

02

Process

I decided to sketch first to develop ideas, then use sketchup 3d to create a digital model, and finally calculate the size to create a 1:10 scale physical model. We only allow using cardboards and Styrofoam (of any thickness) for physical models in the requirements.

First version sketch

This first version of my sketch started with a box-shaped space. Considering the need for users to read and rest in this space, I think there should be some open space for lighting above the box. Then I thought of the time machine in the movie "Back to the future". I think that I can create a circular space and the shape can be designed as a clock. Passers-by on the outside can see the time from the clock, and users on the inside can have their own private space for reading and resting. For siting cube, because the cube does not have to be a cube, I think people sit on chairs that are designed with arcs will be comfortable.

First version physical model

When creating physical models, I encountered challenges in create arcs. I found that making radians directly from cardboard would create creases, and I could n’t create radians using Styrofoam at all. After thinking, I thought that I could choose to use cardboard to create the arc. I can calculate the distance and follow the line space on the cardboard to cut, and then bend to create an arc.

Second version sketch

With the advice of the instructor, I found that my first version of the design gave users limited options for their private. And I should create more open space for air. I should design a sliding window for space. For siting cube, in the first version of the design, I did not consider the multifunctional seat at all. So in the second version of the sketch, I explored the possibility of a multifunctional seat and chose a simple deformation design as my second design.

Second version digital model

In this design, the part of my siting cube uses the tenon structure design, and all parts can be disassembled. When users sleep in space, they can choose to disassemble their seats. When users are reading in space, they can simply combine wooden boards together. When a user needs a platform for storing items, they can disassemble the seat panel and assemble it into the hole in the seat armrest. For the box space, I added holes for ventilation. I also added some sliding windows, so users can choose the amount of light to control their privacy.

Second version physical model

When creating physical models, I encountered the challenge of making sliding windows. After observing the actual sliding windows and thinking, I used toothpicks and cardboard to make the slide rails. I calculated the sliding distance and then used a toothpick as a fixed point at one end to slide the cardboard.

03

Outcome

I think this project is a successful design because I have considered the user's psychology and space utilization, and also learned that the choice should be left to the user. I think this project promotes my thinking on model building, space conception and user experience design, and by watching other people's designs and models, I will also reflect and rebuild my own design, which enhances my critical thinking ability.
